
## 2. 인증 레이어 상세 (패치 후)

```
요청 도착
    │
    ▼
┌──────────────────────────────────────────────────────────┐
│  require_auth() — auth.rs                                │
│                                                          │
│  1. Authorization 헤더 파싱                               │
│     "Bearer did:helm:<base58_pubkey>"                    │
│                                                          │
│  2. DID 레지스트리 조회                                    │
│     agents.contains_key(&did) → 401 if missing           │
│     public_key_b58 추출 (서명 검증용)                     │
│                                                          │
│  3. [✅ C1] ed25519 서명 검증                             │
│     X-Helm-Signature: <base64(ed25519(sha256(...)))>     │
│     - 헤더 있음 + 유효 → 통과                             │
│     - 헤더 있음 + 무효 → 401 Unauthorized                │
│     - 헤더 없음 (write) → 경고 로그 (backward compat)    │
│                                                          │
│  4. [✅ C11] 안티-리플레이 타임스탬프 검증                  │
│     X-Helm-Timestamp: <unix_ms>                          │
│     서명 페이로드: sha256(timestamp_ms_str + ":" + body)  │
│     - 타임스탬프 없음 → 구 프로토콜 (경고)                 │
│     - 타임스탬프 >30s 과거 → 401 signature_expired       │
│     - 타임스탬프 >30s 미래 → 400 timestamp_in_future     │
│                                                          │
│  5. [✅ C6] Rate Limit (슬라이딩 윈도우)                  │
│     state.rate_limits: DID → Vec<timestamp_ms>          │
│     60초 윈도우 내 >100 요청 → 429                        │
│                                                          │
│  6. CallerDid extension 주입                             │
└──────────────────────────────────────────────────────────┘
    │
    ▼
┌──────────────────────────────────────────────────────────┐
│  server.rs 공통 미들웨어 (모든 응답)                        │
│                                                          │
│  [✅ H1] CORS: HELM_CORS_ORIGINS env var allowlist       │
│  [✅ H2] RequestBodyLimitLayer(10MB)                     │
│  [✅ H3] X-Content-Type-Options: nosniff                 │
│          X-Frame-Options: DENY                           │
│  [신규] Strict-Transport-Security: max-age=31536000      │
│  [신규] Cache-Control: no-store                          │
│  [신규] Referrer-Policy: no-referrer                     │
│  [신규] CORS: x-helm-timestamp 헤더 추가                  │
└──────────────────────────────────────────────────────────┘
```

```
방어: X-Helm-Timestamp > 30s 과거 → 401 signature_expired
```
